\section{Goal}

This class project boldly aims to face the challenge of the emotional state recognition in human faces through experimentation of some of the latest techniques available to develop a working prototype of an \emph{emotion detector}, aka \code{Emotime}.

\subsubsection*{Code Base and Other Details}

Project source code and further details are available at \url{https://github.com/luca-m/emotime/}.
